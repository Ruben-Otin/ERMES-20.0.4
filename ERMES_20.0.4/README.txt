This folder contains the ERMES 20.0 graphical user interface and documentation.

To install ERMES 20.0 follow these steps:

1) Download and install GiD (https://www.gidsimulation.com/).

2) Copy&Paste the whole "ERMES_20.0" folder in the "problemtype" folder of GiD.

The folder "\Documents" contains the ERMES 20.0 manual, license information and, 
examples of auxiliary scripts for batch calculations. It also contains examples 
of input files and scripts for the ERMES 20.0 cold plasma module.

The folder "\ERMES.gid" contains the ERMES 20.0 problemtype for GiD and its 
executables for Windows and Linux. 

The folder "\External_Solvers" contains documentation and examples of scripts to
interface ERMES 20.0 with the solvers libraries PETSc and Python NumPy.
